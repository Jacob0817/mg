# mg
### require 
* django 2.1.5  
`$ pip install Django==2.1.5`
* django-users2  
`$ pip install django-users2`
* bootstrap 3.4.0  
`$ npm install bootstrap@3`
* vue 2.5.22  
`$ npm install vue`
* vue-router 3.0.2  
`$ npm install vue-router`
### to run this site
`$ cd /path/to/mg/MgForSchoolV`  
`$ yarn install`  
`$ yarn run build`  
`$ cd /path/to/mg`  
`$ python manage.py runserver`
